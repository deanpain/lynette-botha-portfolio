# Lynette Botha Portfolio

Personal portfolio for **Lynette Botha**, Award-Winning Magazine Editor offering professional publishing, editing, and advertising services.

## Project info

This project showcases Lynette's 15+ years of experience in luxury, travel, lifestyle, and property magazine publishing.

## Features

- **Modern Responsive Design:** Built with React, Tailwind CSS, and Shadcn UI.
- **Service Listings:** Detailed overview of Publishing, Editing, and Advertising Support.
- **Portfolio Gallery:** Showcase of featured magazine titles.
- **Contact Integration:** Easy-to-use inquiry form for potential clients.

## Tech Stack

- **Framework:** React 18 (Vite)
- **Styling:** Tailwind CSS + Lucide Icons
- **Components:** Shadcn/UI (Radix UI)
- **Forms:** React Hook Form + Zod
- **Infrastructure:** TypeScript

## Setup & Development

1. **Install dependencies:**
   ```sh
   npm install
   ```
2. **Start development server:**
   ```sh
   npm run dev
   ```
3. **Build for production:**
   ```sh
   npm run build
   ```

## Deployment

This site is set up for automatic deployment to Vercel via GitHub. Pushing to the `main` branch will trigger a new build.
